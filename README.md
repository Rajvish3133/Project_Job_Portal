**Project Job Portal**

A modern job portal web application that connects job seekers and employers, providing a full-stack system where users can browse jobs, apply, post jobs, manage profiles, and more.


**About**

This is a full-stack Job Portal application designed to help job seekers find relevant employment opportunities and employers to create and manage job postings.

The portal supports:
User authentication and role-based access
Job listing and application
Employer job posting
Profile management


** Features**

✔️ User registration and login (employers & job seekers)
✔ Job search and filter
✔ Apply to job posts
✔ Post job listings (employers)
✔ Profile management
✔ Responsive design (work on desktop & mobile)
✔ Secure authentication and routing
